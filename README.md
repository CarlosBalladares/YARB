# YARB (Yet Another React Boilerplate)

Custom simple and light boilerplate I maintain to bootstrap web projects.

- Supports ES6 syntax
- React
- css modules
- production nd developemnet builds.

## Scripts

To run a development build
`$ yarn dev`

To run a production build
`$ yarn prod`
